CrystalBall¶

class minkit.CrystalBall(name, x, c, s, a, n)[source]¶

Bases: minkit.SourcePDF

Create a new PDF with the name, parameter related to the data and the argument parameters. This PDF is expressed as

\[f\left(x;c,\sigma,\alpha,n\right) = e^{- \frac{\left(x - c\right)^2}{2\sigma^2}}\]

for values of

\[\frac{x - c}{\sigma} \geq - |\alpha|\]

otherwise, the power law

\[\frac{A}{\left(B - \frac{x - c}{\sigma}\right)^n}\]

is used, where

\[A = \left(\frac{n}{|\alpha|}\right)^n e^{- \frac{1}{2} |\alpha|^2}\]
\[B = \frac{n}{|\alpha|} - |\alpha|\]
Parameters

  • name (str) – name of the PDF.

  • x (Parameter) – data parameter.

  • c (Parameter) – center of the Gaussian core.

  • s (Parameter) – standard deviation of the Gaussian core.

  • a (Parameter) – number of standard deviations till the start of the power-law behaviour. A negative value implies the tail is on the right.

  • n (Parameter) – power of the power-law.
