asymmetric_errors(name, cov=None, sigma=1, atol=1e-08, rtol=1e-05, maxcall=None)Â¶

Calculate the asymmetric errors for the given parameter. This is done by subdividing the bounds of the parameter into two till the variation of the FCN is one. Unlike MINOS, this method does not treat new minima.

Parameters

  • name (str) â name of the parameter.

  • cov (numpy.ndarray) â covariance matrix. If provided, the initial values of the parameters will be obtained from them.

  • sigma (float) â number of standard deviations to compute.

  • atol (float) â absolute tolerance for the error.

  • rtol (float) â relative tolerance for the error.

  • maxcall (int or None) â maximum number of calls to calculate each error bound.

Method to ensure that modifications of parameters within a minimizer context are reset properly. Sadly, the iminuit.Minuit class is not stateless, so each time a parameter is modified it must be notified of the change.
