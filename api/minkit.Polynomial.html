Polynomial¶

class minkit.Polynomial(name, x, *coeffs, backend=None)[source]¶

Bases: minkit.SourcePDF

Build the class given the name, parameter related to data and the coefficients. Coefficients must be sorted from lower to higher order. Due to the normalization condition, the first parameter is fixed to zero, if no coefficients are provided the result is a constant function. The PDF is expressed as

\[f\left(x;a_1,a_2,...\right) = 1 + \sum_{i = 1}^n a_i x^i\]
Parameters

  • name (str) – name of the PDF.

  • x (Parameter) – data parameter.

  • coeffs (tuple(Parameter)) – coefficients for the polynomial
