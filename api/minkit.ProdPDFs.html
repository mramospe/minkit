ProdPDFs¶

class minkit.ProdPDFs(name, pdfs)[source]¶

Bases: minkit.pdf_core.MultiPDF

This object represents the product of many PDFs where the data parameters are not shared among them.

Parameters

  • name (str) – name of the PDF.

  • pdfs (list(PDF)) – list of PDFs

evaluate_binned(data, norm_to_data=True)[source]¶

Evaluate the PDF over a binned sample. If “norm_to_data” is provided, then the number of entries is normalized to the sum of values in the data sample. Otherwise it is normalized to the sum of the data values.

Parameters

  • data (BinnedDataSet) – input data.

  • norm_to_data (bool) – possible normalization value.

Returns

values from the evaluation.
