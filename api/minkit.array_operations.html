array_operations¶

class minkit.array_operations(backend, **kwargs)[source]¶

Bases: object

Build the object to do operations within a backend. Only the necessary operators have been defined.

Parameters

  • btype (str) – backend type (‘cpu’, ‘cuda’, ‘opencl’).

  • kwargs – arguments forwarded to the backend constructor (cuda and opencl backends only). It can contain any of the following keys: * device: * interactive: :type kwargs: dict
