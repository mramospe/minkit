fcn_profile¶

minkit.fcn_profile(fcn, pdf, data, wa, values, range='full', constraints=None)[source]¶

Evaluate the profile of an FCN for a set of parameters and values.

Parameters

  • fcn (str) – FCN to use.

  • pdf (PDF) – PDF to evaluate.

  • data (DataSet or BinnedDataSet) – data set.

  • wa (str or list(str)) – single variable or set of variables.

  • values (numpy.ndarray) – values for each parameter specified in wa.

  • range (str) – normalization range for the unbinned case.

  • constraints (list(PDF)) – set of constraints to consider in the minimization.

Returns

Profile of the FCN for the given values.

Return type

numpy.ndarray
